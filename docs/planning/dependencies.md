# 📦 Project Dependencies

Daftar framework dan pustaka utama yang digunakan dalam pengembangan Backend SIMRS.

## 🚀 Framework
- **Fiber v2** (`github.com/gofiber/fiber/v2`): High-performance web framework terinspirasi oleh Express.js. Standar stabil untuk Go >= 1.20.

## ⚙️ Configuration & Logging
- **Viper** (`github.com/spf13/viper`): Solusi konfigurasi lengkap (JSON, TOML, YAML, Env, Flags).
- **Zap** (`go.uber.org/zap`): Logger berperforma tinggi, terstruktur, dan sangat cepat dari Uber.

## 🗄️ Database & Persistence
- **GORM** (`gorm.io/gorm`): ORM untuk Go yang powerful dan lengkap.
- **Postgres Driver** (`gorm.io/driver/postgres`): Driver database PostgreSQL untuk GORM.
- **Golang Migrate** (`github.com/golang-migrate/migrate/v4`): Alat migrasi database berbasis SQL yang versi-aware.

## 🛠️ Utilities
- **UUID** (`github.com/google/uuid`): Generator Universal Unique Identifier.
- **Validator** (`github.com/go-playground/validator/v10`): Go struct data validation.
- **Bcrypt** (`golang.org/x/crypto/bcrypt`): Library untuk hashing password dengan aman.
- **JWT** (`github.com/golang-jwt/jwt/v5`): Implementasi JSON Web Token.
